# 介紹

請先為這個挑戰建立一個資料夾。

執行下面這個命令以建立資料夾，如果有需要，您也可以改變路徑：

`mkdir javascripting`

進入 `javascripting` 資料夾：

`cd javascripting`

建立一個名為 `introduction.js` 的檔案，並且把下面的文字加入檔案：

Add this text to the file:

```js
console.log('hello');
```
  
執行下面的命令，測試你的程式是不是正確地執行：

`javascripting verify introduction.js`
